Applcation is for calculating fizz buzz game(http://codingdojo.org/cgi-bin/index.pl?KataFizzBuzz). 

Solution is provided in the form of maven (http://maven.apache.org/download.cgi) project. 
Application is developed by Java 1.8 language. As a consequence, execution of this application requires the JRE 8 install  (http://www.oracle.com/technetwork/java/javase/downloads/jre8-downloads-1880261.html)

FizzBuzz.java - main class
mvn clean install - builds the application to /target/ directory
mvn clean test - runs junit tests

have a good time :)
